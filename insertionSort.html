Insertion Sort Visualizer

Insertion Sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time. It works by iteratively growing a sorted portion of the array, starting with the first element and continuously adding one more element until the entire array is sorted.

public static void insertionSort(int[] arr) {
    int n = arr.length;
    for(int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}
